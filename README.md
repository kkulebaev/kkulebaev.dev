# kkulebaev.dev

Личный сайт-портфолио на Vue 3 + Vite + TypeScript.

Текущая версия собирается как single-page portfolio в эстетике technical editorial / terminal shell, с упором на tonal layering, sharp corners и минимальное число ad hoc визуальных значений.

## Запуск

```bash
npm install
npm run dev
```

## Проверка production-сборки

```bash
npm run build
```

## Структура

- `src/App.vue` — контентные секции и data-driven структура страницы
- `src/style.css` — foundation tokens, recipes для shell / terminal / buttons / cards и layout страницы
